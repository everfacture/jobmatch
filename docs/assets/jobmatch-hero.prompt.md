# JobMatch Hero Design Brief

This is not an image-generation prompt. The README hero needs controlled text, so the correct artifact is a hand-authored SVG.

## Surface

GitHub README hero/banner for a public local-first CLI tool.

## Design direction

Use a product-card + real-output-preview banner, not abstract AI art.

The banner should show:

- plain project name: `JobMatch`
- tagline: `Find jobs. Rank fit. Send a clean digest.`
- trust/value pills: `Runs locally`, `Bring your own AI`, `Telegram / console / file`, `Dry-run first`
- a small quickstart terminal card with real commands
- a simple pipeline: `Sources -> Score -> Digest`
- a phone/digest card showing ranked scores and job-card shapes
- no fake logos, no decorative AI brain, no cyberpunk dashboard, no made-up SaaS UI

## Why SVG

The asset includes readable text. Image generation is wrong for this because it can garble text. SVG gives controlled typography, exact copy, accessibility text, and clean GitHub rendering.
